Crear un nuevo Nodo

   Si se desea crear un nuevo nodo o vértice se debe hacer click a Edit.

  Luego hacer click a Add Node.

  Luego se debe hacer click dentro del recuadro, a continuación le aparecera una ventana donde   debe ingresar el nombre o identificador y seleccionar si es un Nodo origen o destino.

Crear una nueva Arista

Para crear una nueva arista se debe hacer click a Edit.

Luego se debe hacer click a Add Edge.

Luego debe hacer click al nodo mantienes el click y arrastras hasta el segundo nodo y sueltas.

a continuación le aparecera una ventana donde debe ingresar el valor de la arista.

Opciones de Maximizar o minimizar

Para generar la matriz se debe seleccionar si queremos maximizar los valores o en caso contrario minimizar.

A continuación le aparecera la matriz con los valores encontrados..
